# Talkie

**Talkie** 是一个轻量级、响应迅速且高度可定制的桌面 AI 对话助手。

它是作者作为一个非互联网业内人士，Vibe coding的成果。

---

## 🌟 核心特性

- **极致性能**：基于 Rust (Tauri) 与 Vue 3 构建，极低的内存占用，极快的响应速度。
- **MCP 原生支持**：内置 Model Context Protocol (MCP) 支持，让 AI 能够连接你的本地文件、数据库、搜索引擎等外部工具。
- **深度搜索集成**：支持集成博查 (Bocha)、Tavily 等 AI 搜索引擎，让对话拥有实时获取信息的能力。
- **提示词管理**：内置灵活的提示词模板系统，支持快捷切换和默认设置。
- **隐私优先**：所有配置和聊天记录均本地存储于 SQLite 数据库，不经过任何中转服务器。

---

## 🛠 技术选型

项目在架构设计上追求“小而美”的工业感：
- **后端**：Rust + Tauri，负责系统交互、多进程 MCP 调度及高性能 SSE 解析。
- **前端**：Vue 3 + TypeScript + TailwindCSS + shadcn-vue，提供精致且流畅的交互体验。
- **存储**：SQLite (Rusqlite)，确保数据的持久化与本地安全性。
- **保障**：执行 TDD 流程，拥有覆盖核心逻辑的单元测试与集成测试。

---

## 💡 为什么会有这个项目？

主要因为是我在玩Vibe Coding时对前端掌握非常差，所以选择了一个特别烂大街的项目（聊天机器人）来练手。同时，Talkie 的诞生还源于我对现有 AI 客户端的不适应。

---

## 🤖 开发方式

本项目采用人机协作模式开发：

- **技术选型 / 架构设计 / 视觉设计**：本人主导
- **代码实现**：全部由 AI（OpenCode）完成
- **日志 / 测试 / 方案决策**：人机协作

---

## 🚀 开始使用

### 开发环境准备
- Rust & Cargo
- Node.js & pnpm

### 运行
```bash
# 安装依赖
pnpm install

# 启动开发环境
pnpm tauri dev
```

### 构建
```bash
pnpm tauri build
```
